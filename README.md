## 如何使用

因`github action`出现弱鸡CPU，改为在本地或云服务器上进行编译。

------

## 用户名和密码

 * User: root
 * Password: password
 * Default IP: 10.0.0.11

------

## 全新刷入emmc方法

  1. 固件刷入U盘。
  2. openwrt-install-amlogic
  3. 拔掉U盘，断电重启

------

## 在线升级方法

  1. 上传 img/img.gz 到/mnt/mmcblk2p4
  2. openwrt-update-amlogic
